About

As a Campus Technician at Everman ISD, I help keep infrastructure running across a distributed K-12 network. I work within the district team, show up early, learn fast, and document everything I touch so the next person doesn't have to guess.

My real goal isn't tied to a specific job title. I want stable, meaningful work I can do remotely, with real security in both my job and my skills. That means finishing my B.S. in Cloud and Network Engineering at WGU, holding CompTIA A+ Core 2, and building toward Core 1, Linux, and AWS so I have options.

Outside work I run whatever I can get my hands on. I've operated a World of Warcraft 3.3.5 private server in Docker and experimented with running 500 bots that play the game on their own. I swapped the SSD in my Steam Deck for a larger drive, and I keep a headless Ubuntu box with CasaOS running for the stuff that needs to stay up.
