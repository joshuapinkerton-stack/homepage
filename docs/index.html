About

I keep infrastructure running for a 12-school district in Fort Worth. As a Campus Technician at Everman ISD, my work ranges from pulling cable and patching switch ports to imaging laptops and managing Active Directory accounts. The goal is always the same: make sure teachers can teach and students can learn without tech getting in the way.

I’m also pursuing a B.S. in Cloud and Network Engineering at WGU. I’ve already passed CompTIA A+ Core 2, and I’m working through Core 1 and Linux fundamentals now. Once I finish the degree, I want to move into cloud or data center work, ideally with an AWS focus.

At home I run a lab that mirrors what I see at work: Active Directory, Linux servers, VLANs, and a Minecraft server I use to practice permissions and backups. I treat those setups like real infrastructure, not experiments. That’s where most of my actual learning happens.
